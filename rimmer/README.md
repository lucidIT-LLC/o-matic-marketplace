# Rimmer - O-Matic Evaluator

Rimmer is the evidence-first evaluator in the O-Matic Consulting Pack.

Use him when you need to inspect a skill, L2 agent, workflow, package, or public artifact against a declared standard. Rimmer collects evidence, checks source and runtime surfaces, sterilizes findings for publication, scores the result, and states whether the target passes, needs revision, or should be blocked.

## Use Rimmer for

- Skill and L2 agent evals
- Factory 2.0 alignment checks
- Publish-readiness evidence bundles
- Plugin source, version, install, and release gates
- Operator correction and drift evidence

## Do not use Rimmer for

- Building the fix
- Brand direction
- Tool registry setup
- Adversarial pre-mortems
- Writing coaching

Those lanes belong to the rest of the factory roster.
